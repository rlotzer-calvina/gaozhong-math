# 1.2 集合之间的关系

## 核心讲义

- **子集 (Subset)**：若 $A$ 的任意元素都是 $B$ 的元素，则 $A \subseteq B$。
- **真子集 (Proper Subset)**：若 $A \subseteq B$ 且存在 $x \in B$ 但 $x \notin A$，则 $A \subsetneq B$。
- **集合相等**：若 $A \subseteq B$ 且 $B \subseteq A$，则 $A = B$。
- **空集性质**：
  - $\emptyset$ 是任何集合的子集：$\emptyset \subseteq A$。
  - $\emptyset$ 是任何非空集合的真子集：若 $A \neq \emptyset$，则 $\emptyset \subsetneq A$。
- **子集个数公式**：含 $n$ 个元素的集合，其子集个数为 $2^n$，真子集个数为 $2^n - 1$。

## 精选母题

### 例 6
::: info 题目
若集合 $A=\{x \mid x^2-x-2=0\}$，求 $A$ 的所有真子集。
:::

::: tip 答案
$\emptyset, \{-1\}, \{2\}$
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：** 解方程得 $x_1=-1, x_2=2$，故 $A=\{-1, 2\}$。它的子集有 4 个，剔除它本身 $\{-1, 2\}$ 后剩余上述三个。
**辅导逻辑：** 串联方程求解与真子集排列组合，着重强调不要遗忘空集。
</details>

### 例 7
::: info 题目
已知 $A = \{x \mid 1 < x < a\}$，若 $A = \emptyset$，求 $a$ 的范围。
:::

::: tip 答案
$a \le 1$
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：** 当且仅当区间的右端点小于等于左端点时，不存在任何中间的实数使得条件成立。
**辅导逻辑：** **避坑空集逆向理解法**。让学生理解区间表达式与真实元素的物理意义对应关系。
</details>

### 例 8
::: info 题目
集合 $\{a, b\}$ 的子集个数是多少？
:::

::: tip 答案
4 个
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：** 分别为 $\emptyset, \{a\}, \{b\}, \{a,b\}$。公式为 $2^n$。
**辅导逻辑：** 要求熟记含有 $n$ 个元素的集合其子集总数为 $2^n$，真子集数为 $2^n-1$。
</details>

### 例 13
::: info 题目
设集合 $A = \{1, 3\}$，集合 $B = \{a+1, a^2-1\}$，若 $A \cap B = B$，且 $B$ 非空，求 $a$。
:::

::: tip 答案
$a=2$ 或 $a=0$
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：** 由 $A \cap B = B$ 知 $B \subseteq A$。令 $a+1=1$ 得 $a=0$，此时 $B=\{1, -1\}$ 不满足子集条件，舍去。不断代入其余根并验证发现只有 $a=2$ 与 $a=0$ 的部分情况复合题意，极其考验试错和互异性检验。
**辅导逻辑：** 对参数讨论和互异性的深度融合考量。必须教会他们倒推和反证。
</details>

### 例 19
::: info 题目
已知 $A = \{x \mid -2 \le x \le 5\}$，$B = \{x \mid m+1 \le x \le 2m-1\}$。若 $B \subseteq A$，求 $m$ 的取值范围。
:::

::: tip 答案
$m \le 3$
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：** 
1. 要时刻不忘 $B=\emptyset$ 的可能：即当 $m+1 > 2m-1 \Rightarrow m < 2$ 时，$B$ 为空，必然是 $A$ 的子集。
2. 若 $B \neq \emptyset$，即 $m \ge 2$ 时，必须两端不超界：$m+1 \ge -2$ 且 $2m-1 \le 5$，解出 $2 \le m \le 3$。
两边一结合，得出 $m \le 3$ 全体域。
**辅导逻辑：** **绝对母题：遗忘空集的重灾区**。如果学生只做出 $2 \le m \le 3$，必定是没有对子基线进行空集保护判断。
</details>

### 例 20
::: info 题目
已知集合 $A = \{x \mid x^2 - 3x + 2 = 0\}$, $B = \{x \mid x^2 - ax + 2 = 0\}$，若 $B \subseteq A$，求实数 $a$ 的取值范围。
:::

::: tip 答案
$a \in (-2\sqrt{2}, 2\sqrt{2}) \cup \{3\}$
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：**
我们分为两步进行深挖：
1. **防坑（考虑空集）**：
因为 $B \subseteq A$，自然包含 $B = \emptyset$ 这种特例。
当二次方程无实数根时，$B$ 为空集，必定落入 $A$ 内。
利用判别式：$\Delta = (-a)^2 - 4(1)(2) < 0 \implies a^2 - 8 < 0$
得到空集条件下的参数解： $-2\sqrt{2} < a < 2\sqrt{2}$。

2. **检验（非空子集）**：
原集合解得 $A=\{1, 2\}$。若 $B \neq \emptyset$，则 $B$ 中元素必源于 $A$。
由韦达定理，方程 $x^2 - ax + 2 = 0$ 的两根之积必然等于 2（即 $x_1 x_2 = 2$）。
此时如果在集合 A 里挑，只能是 $1$ 和 $2$ 分别成为了方程的两个解。
因而其根必然为 $1$ 与 $2$ 这一对。
再用韦达定理两根之和追溯回参数 $a$：
$x_1 + x_2 = a \implies 1 + 2 = a \implies a = 3$

最后，综上两块逻辑体系取并集。
**故实数 $a$ 的收敛范围为：$(-2\sqrt{2}, 2\sqrt{2}) \cup \{3\}$**。

**辅导逻辑：** 
本题为集合子集理论与微元方程判别式深度结合的经典压轴主考大坑，**常考查空集的隐蔽性和韦达定理的使用！** 必须要逼迫考生在一遇到参数挂在自变量上的题目时立刻浮现出三段论式：空集不？互异不？超边不？只有如此方能彻底扫清集合板块的所有选择题诡雷！
</details>
