# 1.1 集合的概念与表示

## 核心讲义

- **集合的定义**：一些确定的、相互区别的对象构成的整体。
- **集合三要素**：
  - **确定性**：集合中的元素必须是确定的。
  - **互异性**：集合中的元素必须互不相同。
  - **无序性**：集合中的元素没有先后顺序。
- **常用数集符号**：
  - $\mathbb{N}$：自然数集（包含 0）
  - $\mathbb{N}^*$ 或 $\mathbb{N}_+$：正整数集（不含 0）
  - $\mathbb{Z}$：整数集
  - $\mathbb{Q}$：有理数集
  - $\mathbb{R}$：实数集

## 精选母题

### 例 1
::: info 题目
已知集合 $A = \{1, 2, 3\}$，$B = \{2, 3, 4\}$，求 $A \cup B$。
:::

::: tip 答案
$\{1, 2, 3, 4\}$
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：** 将两集合合并，根据互异性原则，重复的 2 和 3 只写一次。
**辅导逻辑：** 考核并集基础概念。并集指所有属于 A 或 B 的元素集合，务必强调元素的互异性。
</details>

### 例 4
::: info 题目
判断关系：$0 \in \mathbb{N}$ 是否正确？
:::

::: tip 答案
正确
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：** 自然数集 $\mathbb{N}$ 是包含非负整数的，因此 $0$ 属于自然数集。
**辅导逻辑：** 澄清概念混淆点。许多人会将 $\mathbb{N}$ 和 $\mathbb{N}^*$ (正整数集) 记混，这是常见选择题陷阱。
</details>

### 例 12
::: info 题目
已知 $M = \{(x,y) \mid x+y=2\}$，$N = \{(x,y) \mid x-y=4\}$，求 $M \cap N$。
:::

::: tip 答案
$\{(3, -1)\}$
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：** 这是两直线的点集求交，等同于解二元一次方程组 $x+y=2, x-y=4$。解出 $x=3, y=-1$。因元素的本质坐标对，外层不应遗弃大括号。
**辅导逻辑：** 点集的集合形式。着重批评部分学生写成 $\{3, -1\}$（一维错误）的顽疾。
</details>

### 例 14
::: info 题目
若集合 $A=\{x \mid ax+1=0\}$ 中只有一个元素，求 $a$。
:::

::: tip 答案
$a \neq 0$
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：** 当 $a=0$ 的时候方程无解，对应空集；只要 $a \neq 0$，就是标准且具有单解的一次方程。
**辅导逻辑：** 这是后续解析数论里讨论二次方程一次化降维打击的雏形。
</details>

### 例 15
::: info 题目
$A=\{1,2,3,4,5\}$ 中能被 2 整除的元素构成的子集？
:::

::: tip 答案
$\{2, 4\}$
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：** 能被 2 整除即为偶数，在基础数组里直接挑出筛选。
**辅导逻辑：** 了解条件约束筛选子集的基础手段。
</details>
