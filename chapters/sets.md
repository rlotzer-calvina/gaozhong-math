# 第一章：集合与常用逻辑用语 (复习全能指南-V5终极版)

## 一、核心概念与关系

- **集合三要素**：确定性、互异性、无序性。
- **常用数集符号**：
  - $\mathbb{N}$ (自然数集), $\mathbb{N}^*$ 或 $\mathbb{N}_+$ (正整数集)
  - $\mathbb{Z}$ (整数集)
  - $\mathbb{Q}$ (有理数集)
  - $\mathbb{R}$ (实数集)
- **属于/包含关系**：
  - 元素与集合：用 $\in$ 或 $\notin$ 表达。
  - 集合与集合：用 $\subseteq$ 或 $\subsetneq$ 表达。
- **空集性质**：$\emptyset$ 是没有任何元素的集合，规定 $\emptyset$ 是任何集合的子集，即 $\emptyset \subseteq A$。

## 二、集合运算定律

- **交换律**：$A \cup B = B \cup A$, $A \cap B = B \cap A$
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$, $(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$, $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- **德·摩根定律 (De Morgan's Laws)**：
  配合记忆口诀：“长杠变短杠，符号变方向”。
  $$\complement_U (A \cup B) = (\complement_U A) \cap (\complement_U B)$$
  $$\complement_U (A \cap B) = (\complement_U A) \cup (\complement_U B)$$

## 三、三大避坑指南

1. ⚠️ **遗忘空集**：在求解包含参数关系的 $A \subseteq B$ 时，极其容易忽略子集 $A = \emptyset$ 这个先天恒成立的条件。
2. ⚠️ **忽视互异性**：在求出含参数的集合元素后，若不将参数代回原集合检验，可能导致集合中出现两个相同的元素，违背互异性原则。
3. ⚠️ **代表元素错误**：看错集合代表的意义。例如 $A=\{x \mid y=x^2\}$ 是定义域；$B=\{y \mid y=x^2\}$ 是值域；$C=\{(x,y) \mid y=x^2\}$ 是图像上的点集。三者本质截然不同！

## 四、1.4 模块：充分条件与必要条件

- **核心定义**：
  若 $p \Rightarrow q$，即从 $p$ 能够直接推出 $q$，则称 $p$ 是 $q$ 的**充分条件**。而 $q$ 称为 $p$ 的**必要条件**。
  若 $p \Rightarrow q$ 且 $q \Rightarrow p$（即 $p \Leftrightarrow q$），则称 $p$ 是 $q$ 的**充要条件**（充分且必要条件）。
- **判定技巧（集合法）**：
  设满足条件 $p$ 的对象集合为 $A$，满足条件 $q$ 的对象集合为 $B$：
  - **小集合推大集合**：若 $A \subseteq B$，则 $p \Rightarrow q \Leftrightarrow A \text{ 是 } B \text{ 的充分条件}$。
  - 若 $A \subsetneq B$，则 $p$ 是 $q$ 的**充分不必要条件**。
  - 若 $A \supsetneq B$，则 $p$ 是 $q$ 的**必要不充分条件**。
  - 若 $A = B$，则 $p$ 是 $q$ 的**充要条件**。

::: tip 提炼口诀
**判定口诀：小推大，充分；大推小，必要。**
即 $A \subseteq B \iff A \Rightarrow B$（充分）。
:::

## 五、1.5 模块：全称量词与存在量词（逻辑衔接）

| 名称 | 符号 | 读法 | 否定形式 |
| :--- | :--- | :--- | :--- |
| **全称量词** | $\forall$ | 对所有、对任意、一切 | 存在、至少有一个 ($\exists$) |
| **存在量词** | $\exists$ | 存在、有一个、至少有一个 | 所有、任意、一切 ($\forall$) |

- **全称命题**：含有全称量词“所有”、“任意” (符号 $\forall$) 的命题。
  - 标准格式：$\forall x \in M, p(x)$
  - 否定格式：$\neg(\forall x \in M, p(x)) \Leftrightarrow \exists x \in M, \neg p(x)$ （变 $\forall$ 为 $\exists$，并否定结论）
- **特称命题 (存在命题)**：含有存在量词“存在”、“至少有一个” (符号 $\exists$) 的命题。
  - 标准格式：$\exists x \in M, p(x)$
  - 否定格式：$\neg(\exists x \in M, p(x)) \Leftrightarrow \forall x \in M, \neg p(x)$ （变 $\exists$ 为 $\forall$，并否定结论）

## 六、1.6 模块：反证法

反证法是一种间接证明逻辑，特别适用于正面推导极度困难的命题（如“无理数判定”、“唯一性”、“至少/至多存在几个”）。

::: info 步骤
1. **反设**：假设命题结论不成立（即假设结论的反面成立）。
2. **归谬**：从反面假设出发，经过缜密且符合逻辑的推导，得出明显的**矛盾**（可以是与前置已知条件、数学公理、或公认规律相违背）。
3. **结论**：因为推导出逻辑悖论点，故反证由于假设出错导致，从而否定假设，肯定原命题。
:::

### 例 19
::: info 题目
已知 $A = \{x \mid -2 \le x \le 5\}$，$B = \{x \mid m+1 \le x \le 2m-1\}$。若 $B \subseteq A$，求 $m$ 的取值范围。
:::

::: tip 答案

:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

两边一结合，得出 $m \le 3$ 全体域。
**辅导逻辑：** **绝对母题：遗忘空集的重灾区**。如果学生只做出 $2 \le m \le 3$，必定是没有对子基线进行空集保护判断。

### 例 20
::: info 题目
已知集合 $A = \{x \mid x^2 - 3x + 2 = 0\}$, $B = \{x \mid x^2 - ax + 2 = 0\}$，若 $B \subseteq A$，求实数 $a$ 的取值范围。
:::

::: tip 答案
$a \in (-2\sqrt{2}, 2\sqrt{2}) \cup \{3\}$
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：**
我们分为两步进行深挖：
1. **防坑（考虑空集）**：
因为 $B \subseteq A$，自然包含 $B = \emptyset$ 这种特例。
当二次方程无实数根时，$B$ 为空集，必定落入 $A$ 内。
利用判别式：$\Delta = (-a)^2 - 4(1)(2) < 0 \implies a^2 - 8 < 0$
得到空集条件下的参数解： $-2\sqrt{2} < a < 2\sqrt{2}$。

2. **检验（非空子集）**：
原集合解得 $A=\{1, 2\}$。若 $B \neq \emptyset$，则 $B$ 中元素必源于 $A$。
由韦达定理，方程 $x^2 - ax + 2 = 0$ 的两根之积必然等于 2（即 $x_1 x_2 = 2$）。
此时如果在集合 A 里挑，只能是 $1$ 和 $2$ 分别成为了方程的两个解。
因而其根必然为 $1$ 与 $2$ 这一对。
再用韦达定理两根之和追溯回参数 $a$：
$x_1 + x_2 = a \implies 1 + 2 = a \implies a = 3$

最后，综上两块逻辑体系取并集。
**故实数 $a$ 的收敛范围为：$(-2\sqrt{2}, 2\sqrt{2}) \cup \{3\}$**。

**辅导逻辑：** 
本题为集合子集理论与微元方程判别式深度结合的经典压轴主考大坑，**常考查空集的隐蔽性和韦达定理的使用！** 必须要逼迫考生在一遇到参数挂在自变量上的题目时立刻浮现出三段论式：空集不？互异不？超边不？只有如此方能彻底扫清集合板块的所有选择题诡雷！
</details>

### 例 21
::: info 题目
条件 $p: x > 2$，条件 $q: x^2 - 3x + 2 > 0$，则 $p$ 是 $q$ 的什么条件？
:::

::: tip 答案
充分不必要条件
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：**
解不等式 $x^2 - 3x + 2 > 0$，得到 $q$ 代表的集合为 $B = \{x \mid x > 2 \text{ 或 } x < 1\}$。
定义 $p$ 代表的集合为 $A = \{x \mid x > 2\}$。
观察可知 $A \subsetneq B$，由于成立“小集合推大集合”定则，故有 $p \Rightarrow q$ 成立，但 $q \Rightarrow p$ 不成立。
结论为 $p$ 是 $q$ 的**充分但不必要条件**。
**辅导逻辑：**
这是针对 **“1.4 充要条件判定”** 的经典考核。采用“集合法”，将冰冷的不等式转换为直观的集合，通过辨别两集合的真子集包含关系，迅速无误地进行充分性和必要性的靶向判定。
</details>

### 例 22
::: info 题目
设 $x, y \in \mathbb{R}$，试判断“$x=y$”是“$x^2=y^2$ 且 $x^3=y^3$”的什么条件。
:::

::: tip 答案
充要条件 ($p \Leftrightarrow q$)
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：**
先判断充分性 ($p \Rightarrow q$)：若 $x=y$，显然两侧各自取二次方与三次方均相等，故充分性完美成立。
再判断必要性 ($q \Rightarrow p$)：若 $x^2=y^2$ 且 $x^3=y^3$ 成立。由于 $x^2=y^2$，则存在 $x=y$ 或 $x=-y$。
- 若 $x=y$，已经符合结论。
- 若 $x=-y$，代入 $x^3=y^3$ 后得到 $(-y)^3 = y^3 \implies -y^3 = y^3 \implies 2y^3 = 0 \implies y=0$。此时 $x=-0=0$，所以依然有 $x=y$。在任何分歧路线上必定回归 $x=y$。故必要性成立。
两者等效互证，所以是充要条件。
**辅导逻辑：**
用于演练对于复合多元命题进行严谨“左推右”和“右推左”分离演算的典范。必须让学生杜绝靠直觉双向同时预判的坏习惯。
</details>

### 例 23
::: info 题目
应用反证法证明核心特例命题：$\sqrt{2}$ 是无理数。
:::

::: tip 答案
见详细解析
:::

<details>
<summary>点击查看详细解析与辅导逻辑</summary>

**详细解析：**
本题使用**1.6 反证法**三大标准操作规程：
1. **反设**：假设命题不成立，即假设 $\sqrt{2}$ 不是无理数，而是**有理数**。
2. **归谬**：既然是有理数，根据定义，它必可以表示成为一个既约分数 $\sqrt{2} = \frac{a}{b}$ （这里 $a, b$ 是正整数且**二者互质**，没有 1 以外的公约数）。
对方程两端直接平方法：$2 = \frac{a^2}{b^2} \implies a^2 = 2b^2$。
上式证明了 $a^2$ 必是偶数。那么其底数 $a$ 本身必定也是偶数。于是我们可设 $a = 2k$ ($k$ 为正整数)。
将其代入原式得到：$(2k)^2 = 2b^2 \implies 4k^2 = 2b^2 \implies b^2 = 2k^2$。
同样的逻辑，这证明了 $b^2$ 是偶数并得出：$b$ 也是偶数。
若 $a$ 和 $b$ 都是偶数，他们就会存在一个公约数 2，这无情地打破并**矛盾**于开局的前提边界条件 “$a, b$ 二者互约互质”！
3. **结论**：既然导致了公理层逻辑的绝大矛盾，原发假设显然彻底崩塌。故肯定原命题，$\sqrt{2}$ 是严谨且真实的无理数。
**辅导逻辑：**
反证法（1.6 模块）的黄金启蒙母题。以此揭示了什么叫做“无路可走时的退守反攻”。对唯一性或者不含等词特性的命题时常爆发奇效！
</details>
