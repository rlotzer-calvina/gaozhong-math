# 1.4 集合的运算

## 一、交集

由属于集合 $A$ **且** 属于集合 $B$ 的所有元素组成的集合，称为 $A$ 与 $B$ 的**交集**，记作 $A \cap B$。

$$A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$$

::: details 点击展开图示

```
    ╭───────────────────╮
   ╱    A                ╲
  │   ╭────────╮         │
  │  ╱ (A∩B)   ╲         │
  │  │ ■■■■■■   │         │
  │  ╲  (交集)  ╱          │
  │   ╰────────╯         │
  ╰──────────────────────╯

  A∩B：用■标出的阴影区域
```
:::

::: tip 记忆口诀
"交"就是两者**都有**的元素。
:::

**运算律：**
- $A \cap A = A$
- $A \cap \varnothing = \varnothing$
- $A \cap B = B \cap A$（交换律）
- $(A \cap B) \cap C = A \cap (B \cap C)$（结合律）

## 二、并集

由属于集合 $A$ **或** 属于集合 $B$ 的所有元素组成的集合，称为 $A$ 与 $B$ 的**并集**，记作 $A \cup B$。

$$A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$$

::: details 点击展开图示

```
    ╭───────────────────╮
   ╱    A                ╲
  │   ╭────────╮         │
  │  ╱ ■■■■■■■■ ╲         │
  │  │ ■■■■■■   │         │
  │  ╲ ■■■■■■■■ ╱          │
  │   ╰────────╯         │
  ╰──────────────────────╯

  A∪B：A和B的所有区域
```
:::

::: tip 记忆口诀
"并"就是两者**合在一起**的所有元素（去重）。
:::

**运算律：**
- $A \cup A = A$
- $A \cup \varnothing = A$
- $A \cup B = B \cup A$（交换律）
- $(A \cup B) \cup C = A \cup (B \cup C)$（结合律）
- $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$（分配律）
- $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$（分配律）

## 三、补集

在研究某个问题时，如果集合 $I$ 包含了我们所考虑的所有元素，则称 $I$ 为**全集**。

对于全集 $I$ 中的任意集合 $A$，$A$ 的**补集**（或**余集**）定义为：

$$\complement_I A = \{x \in I \mid x \notin A\}$$

::: tip 通俗理解
补集就是"在全集里把 $A$ 挖掉"剩下的部分。
:::

::: details 点击展开图示

```
    ╭───────────────────╮
   ╱       I (全集)       ╲
  │                       │
  │   ╭──────────╮       │
  │  ╱   A (已挖)  ╲      │
  │  │             │      │
  │  ╲             ╱      │
  │   ╰──────────╯       │
  │   （A以外的区域）      │
  ╰──────────────────────╯

  补集 = 全集 I 中不属于 A 的部分
```
:::

## 四、交、并、补运算的综合

| 运算 | 定义 | 关键词 |
|------|------|--------|
| 交集 $A \cap B$ | 既在 $A$ 又在 $B$ | "且"、"and" |
| 并集 $A \cup B$ | 在 $A$ 或 $B$ 或两者 | "或"、"or" |
| 补集 $\complement A$ | 不在 $A$ 中 | "非"、"不是" |

## 五、常用结论

| 公式 | 说明 |
|------|------|
| $A \cap B \subseteq A \subseteq A \cup B$ | — |
| $A \cap B \subseteq B \subseteq A \cup B$ | — |
| $A \cap B = \varnothing$ | $A$ 与 $B$ 无公共元素 |
| $A \cup B = I$ 且 $A \cap B = \varnothing$ | $B$ 是 $A$ 的补集 |
| $\complement(\complement A) = A$ | 补集的补集是自身 |
| $\complement(A \cup B) = \complement A \cap \complement B$ | 德·摩根定律 |
| $\complement(A \cap B) = \complement A \cup \complement B$ | 德·摩根定律 |

::: warning 德·摩根定律（重要）
- $\overline{A \cup B} = \overline{A} \cap \overline{B}$
- $\overline{A \cap B} = \overline{A} \cup \overline{B}$

口诀：**加号变乘号，开门变关门**。
:::

## 六、例题

::: details 点击查看例题

**例1：** 设 $A = \{1, 2, 3\}$，$B = \{2, 4, 6\}$，求 $A \cap B$、$A \cup B$。

**解析：**

$A \cap B = \{2\}$（只有 2 是两者共有的）

$A \cup B = \{1, 2, 3, 4, 6\}$（合并，去重）

---

**例2：** 设全集 $I = \{1, 2, 3, 4, 5\}$，$A = \{1, 3\}$，求 $\complement_I A$。

**解析：**

$\complement_I A = \{2, 4, 5\}$（在全集 $I$ 中去掉 1 和 3）

---

**例3：** 已知 $A = \{x \mid x > 0\}$，$B = \{x \mid x < 3\}$，求 $A \cap B$、$A \cup B$。

**解析：**

$A \cap B = \{x \mid 0 < x < 3\}$，即 $(0, 3)$

$A \cup B = \{x \mid x > 0 \text{ 或 } x < 3\}$，即全体实数 $\mathbb{R}$（因为 $x > 0$ 或 $x < 3$ 覆盖了所有实数）

---

**例4：** 证明德·摩根定律：$\complement(A \cup B) = \complement A \cap \complement B$。

**解析：**

设 $x \in \complement(A \cup B)$，则 $x \notin A \cup B$，即 $x \notin A$ 且 $x \notin B$。

所以 $x \in \complement A$ 且 $x \in \complement B$，即 $x \in \complement A \cap \complement B$。

故 $\complement(A \cup B) \subseteq \complement A \cap \complement B$。

同理可证反向包含，故两边相等。证毕。

:::
