# 1.1 集合的概念

## 一、集合的定义

**集合**：把一些能够确定的、不同的对象汇集在一起，形成的整体。

- 这些对象称为集合的**元素**
- 集合通常用大写字母 $A, B, C, \cdots$ 表示
- 元素通常用小写字母 $a, b, c, \cdots$ 表示

## 二、元素与集合的关系

对于元素 $a$ 与集合 $A$：

- 若 $a$ 在集合 $A$ 中，称 $a$ **属于** $A$，记作 $a \in A$
- 若 $a$ 不在集合 $A$ 中，称 $a$ **不属于** $A$，记作 $a \notin A$

::: tip 记忆口诀
属于 / 不属于，看元素是否在集合"里面"。
:::

## 三、集合的特性

| 特性 | 说明 |
|------|------|
| **确定性** | 任意元素是否属于集合是明确无误的 |
| **互异性** | 集合中的元素互不重复 |
| **无序性** | 集合中的元素没有顺序之分 |

## 四、常用数集

| 符号 | 集合名称 | 包含的数 |
|------|---------|---------|
| $\mathbb{N}$ | 自然数集 | $0, 1, 2, 3, \cdots$ |
| $\mathbb{N}^*$ 或 $\mathbb{N}_+$ | 正整数集 | $1, 2, 3, \cdots$ |
| $\mathbb{Z}$ | 整数集 | $\cdots, -2, -1, 0, 1, 2, \cdots$ |
| $\mathbb{Q}$ | 有理数集 | 整数、分数（有限小数或循环小数） |
| $\mathbb{R}$ | 实数集 | 有理数 + 无理数 |

::: warning 注意
沪教版的自然数集 $\mathbb{N}$ 通常包含 $0$，需留意题目要求。
:::

## 五、例题

::: details 点击查看例题

**例1：** 判断下列对象能否组成集合，并说明理由。

（1）大于 5 的整数；　（2）世界上最高的山；　（3）的好朋友。

**解析：**
- （1）可以组成集合 —— 每个整数是否大于 5 是确定的
- （2）可以组成集合 —— 世界上最高的山是确定的（珠穆朗玛峰）
- （3）不能组成集合 —— "漂亮"没有确定的标准，因人而异

:::
