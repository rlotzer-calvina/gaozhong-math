# 1.5 Venn图与容斥原理

## 一、Venn图（维恩图）

Venn图是用封闭曲线（通常用圆）表示集合的图形工具，能直观地展示集合之间的关系与运算。

### 基本表示

| 图形 | 含义 |
|------|------|
| 一个圆表示集合 $A$ | $A$ 的所有元素在圆内 |
| 圆外区域 | 不属于 $A$ 的元素 |
| 两圆相交 | 交集 $A \cap B$ |
| 两圆合并 | 并集 $A \cup B$ |

### 常见运算图形

```
  A∩B              A∪B               A的补集

  ╭───╮           ╭───╮
 ╱ ╲ █ ╱ ╲        ╱   ╲   ╱
│  ██  │        ╱  ██  ╲
 ╲ ╱ █ ╲ ╱         ╲   ╱   ╲
  ╰───╯           ╰───╯
```

## 二、容斥原理

容斥原理是计算有限个集合并集元素个数的公式。

### 两个集合的容斥原理

$$|A \cup B| = |A| + |B| - |A \cap B|$$

::: tip 含义
先相加两个集合的元素个数，其中交集部分被重复计算了一次，所以要减去一次。
:::

::: details 图示理解

```
    ╭─────────────────╮
   ╱      A∪B         ╲
  │   ╭─────────╮     │
  │  ╱ ■■■■■■■■  ╲     │
  │  │ ■■■│■■■■■ │     │  A 有 5 个元素
  │  │ ■■■│     │     │  B 有 4 个元素
  │  ╲ ■■■│■■■■■ ╱     │  A∩B 有 2 个元素
  │   ╰─────────╯     │
  ╰────────────────────╯

  5 + 4 - 2 = 7（A∪B 的元素个数）
```
:::

### 三个集合的容斥原理

$$|A \cup B \cup C| = |A| + |B| + |C| - |A \cap B| - |B \cap C| - |A \cap C| + |A \cap B \cap C|$$

::: tip 记忆口诀
"奇数个集合 = 加，偶数个集合 = 减"
:::

## 三、应用场景

容斥原理常用于解决以下问题：

- 统计同时满足多个条件的元素个数
- 调查问卷中满足条件的人数统计
- 编程中集合去重计算

## 四、例题

::: details 点击查看例题

**例1：** 某班有 40 人，数学竞赛有 25 人参加，物理竞赛有 20 人参加，两项都参加的有 10 人。问：有多少人两项都不参加？

**解析：**

设 $A$ = 参加数学竞赛的人，$B$ = 参加物理竞赛的人。

$|A| = 25$，$|B| = 20$，$|A \cap B| = 10$

$|A \cup B| = 25 + 20 - 10 = 35$（至少参加一项的人数）

两项都不参加 = $40 - 35 = 5$ 人

---

**例2：** 某校高一有 200 人，其中喜欢足球的有 120 人，喜欢篮球的有 80 人，两项都喜欢的有 30 人。求只喜欢足球的有多少人？只喜欢篮球的有多少人？两项都不喜欢的有多少人？

**解析：**

只喜欢足球 = $|A| - |A \cap B| = 120 - 30 = 90$ 人

只喜欢篮球 = $|B| - |A \cap B| = 80 - 30 = 50$ 人

至少喜欢一项 = $120 + 80 - 30 = 170$ 人

两项都不喜欢 = $200 - 170 = 30$ 人

---

**例3（三个集合）：** 某班 50 人，订阅《数学报》的有 30 人，订阅《语文报》的有 25 人，订阅《英语报》的有 20 人。同时订阅《数学》和《语文》的有 10 人，同时订阅《语文》和《英语》的有 8 人，同时订阅《数学》和《英语》的有 5 人，三种都订阅的有 3 人。问三种都不订阅的有多少人？

**解析：**

设 $A, B, C$ 分别为订阅《数学》《语文》《英语》的学生集合。

$|A| = 30, |B| = 25, |C| = 20$
$|A \cap B| = 10, |B \cap C| = 8, |A \cap C| = 5$
$|A \cap B \cap C| = 3$

$|A \cup B \cup C| = 30 + 25 + 20 - 10 - 8 - 5 + 3 = 55$

三种都不订阅 = $50 - 55 = -5$（人数不能为负，说明题目数据有误，实际中应检查数据）

:::
