# 1.2 集合的表示方法

## 一、列举法

将集合中的所有元素一一列举出来，写在大括号 {} 内。

$$A = \{1, 2, 3, 4, 5\}$$

**适用场景**：元素个数有限且不多时。

::: tip 示例
- 不大于 10 的正偶数：$B = \{2, 4, 6, 8, 10\}$
- 方程 $x^2 - 5x + 6 = 0$ 的解：$C = \{2, 3\}$
:::

## 二、描述法

将集合中元素所具有的共同特征或满足的条件描述出来。

$$A = \{x \in \mathbb{R} \mid x^2 - 4 = 0\}$$

解读：大括号内，竖线左侧写元素的一般形式，竖线右侧写元素满足的条件。

::: tip 示例
- 不等式 $2x - 1 > 3$ 的解：$\{x \in \mathbb{R} \mid x > 2\}$
- 沪教版教材第 1-3 页例题：$\{x \mid x \text{ 是正方形}\}$
- 方程 $x^2 = 9$ 的解：$\{x \in \mathbb{Z} \mid x^2 = 9\}$（整数解）
:::

## 三、图示法（Venn图）

用平面上一条封闭曲线的内部表示集合。

::: tip 示例
用 Venn 图表示集合 $A = \{1, 2, 3\}$：

```
        ╭───────╮
       ╱  1  2  ╲
      │  3      │
       ╲        ╱
        ╰───────╯
```

:::

## 四、区间表示法

用于表示数轴上连续的实数集合。

| 写法 | 含义 | 数轴表示 |
|------|------|---------|
| $(a, b)$ | $a < x < b$ | 不含端点的线段 |
| $[a, b]$ | $a \leqslant x \leqslant b$ | 含端点的线段 |
| $(a, b]$ | $a < x \leqslant b$ | 左开右闭 |
| $[a, b)$ | $a \leqslant x < b$ | 左闭右开 |
| $(a, +\infty)$ | $x > a$ | 向右的射线 |
| $(-\infty, a]$ | $x \leqslant a$ | 向左的射线 |

## 五、自然语言与集合语言的转换

| 自然语言 | 集合语言 |
|---------|---------|
| $x$ 大于 3 | $x > 3$ |
| $x$ 是正整数 | $x \in \mathbb{N}^*$ |
| $x$ 属于集合 $A$ | $x \in A$ |
| $x$ 不小于 5 | $x \geqslant 5$ |
| $x$ 在 2 和 10 之间 | $2 < x < 10$ |
