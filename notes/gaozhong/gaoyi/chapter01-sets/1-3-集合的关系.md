# 1.3 集合的关系

## 一、子集

若集合 $A$ 中的**每一个**元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作：

$$A \subseteq B \quad \text{或} \quad B \supseteq A$$

读作：$A$ 包含于 $B$（或 $B$ 包含 $A$）

::: tip 通俗理解
"小集合"在大集合"里面"。
:::

$A$ 的所有元素都在 $B$ 内，则 $A \subseteq B$。
:::

## 二、真子集

若 $A \subseteq B$，且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作：

$$A \subsetneqq B \quad \text{或} \quad B \supsetneqq A$$

::: tip 子集 vs 真子集
- **子集**：$A$ 的元素全在 $B$ 中（$A$ 可以等于 $B$）
- **真子集**：$A$ 的元素全在 $B$ 中，且 $A$ 严格小于 $B$
:::

## 三、空集

**空集**：不含有任何元素的集合，记作 $\varnothing$。

::: warning 重要性质
- 空集是**任何集合**的子集：$\varnothing \subseteq A$（对任意集合 $A$）
- 空集是**任何非空集合**的真子集：$\varnothing \subsetneqq A$（$A \neq \varnothing$）
- 空集本身是自身的子集：$\varnothing \subseteq \varnothing$
:::

## 四、集合相等

若 $A \subseteq B$ 且 $B \subseteq A$，则称 $A$ 与 $B$ **相等**，记作 $A = B$。

::: tip 证明集合相等的方法
要证 $A = B$，只需证明 $A \subseteq B$ 且 $B \subseteq A$。
:::

## 五、常用结论

| 结论 | 说明 |
|------|------|
| $A \subseteq A$ | 集合与自身相等（反身性） |
| 若 $A \subseteq B$，$B \subseteq C$，则 $A \subseteq C$ | 传递性 |
| $\varnothing \subseteq A$ | 空集是所有集合的子集 |
| 若 $A \neq B$，则 $A \not\subseteq B$ 或 $B \not\subseteq A$ | — |
| 含 $n$ 个元素的集合，有 $2^n$ 个子集 | 包括自身和空集 |

## 六、含 $n$ 个元素的集合

若集合 $A$ 有 $n$ 个元素，则：

| 子集类型 | 个数 |
|---------|------|
| 子集（含真子集） | $2^n$ |
| 真子集（非空真子集） | $2^n - 1$ |
| 非空子集 | $2^n - 1$ |

::: tip 示例
集合 $\{1, 2, 3\}$ 有 $n = 3$ 个元素：
- 子集有 $2^3 = 8$ 个：$\varnothing, \{1\}, \{2\}, \{3\}, \{1,2\}, \{1,3\}, \{2,3\}, \{1,2,3\}$
- 真子集有 $2^3 - 1 = 7$ 个（去掉 $\{1,2,3\}$ 自身）
:::

## 七、例题

::: details 点击查看例题

**例1：** 判断以下关系是否正确，并说明理由。

（1）$\{1, 2\} \subseteq \{1, 2, 3\}$
（2）$\{1, 2\} \subsetneqq \{1, 2, 3\}$
（3）$\varnothing \subseteq \{1\}$
（4）$\varnothing = \{0\}$

**解析：**

（1）正确，$\{1, 2\}$ 的所有元素都在 $\{1, 2, 3\}$ 中。

（2）正确，且 $\{1, 2\} \neq \{1, 2, 3\}$，所以是真子集。

（3）正确，空集是所有集合的子集。

（4）错误，$\varnothing$ 不含任何元素，$\{0\}$ 含有一个元素 $0$，两者不相等。

---

**例2：** 已知集合 $A = \{x \mid x^2 = 4\}$，$B = \{-2, 2\}$，判断 $A$ 与 $B$ 的关系。

**解析：**

$A = \{x \mid x^2 = 4\} = \{-2, 2\}$，与 $B$ 的元素完全相同。

所以 $A = B$，同时也满足 $A \subseteq B$ 和 $B \subseteq A$。

---

**例3：** 集合 $\{a, b\}$ 有多少个子集？分别写出来。

**解析：**

$n = 2$，子集个数为 $2^2 = 4$ 个：

$\varnothing, \{a\}, \{b\}, \{a, b\}$

:::
